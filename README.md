# htmx-lsp

Language Server for [htmx](https://htmx.org)


## Installation
You need [Node.js](https://nodejs.org/) installed on your machine.

Install `htmx-lsp` globally
```
  npm i -g htmx-lsp
```

- Autocomplete for htmx attribute names with documentation
